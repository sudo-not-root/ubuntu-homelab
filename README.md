# Ubuntu Homelab

## Overview

This repository documents the development of my personal Ubuntu-based cybersecurity homelab. The purpose of this project is to strengthen practical skills in Linux administration, networking, cybersecurity operations, scripting, containerization, and AI-assisted workflows through hands-on experimentation and documentation.

The homelab serves as an environment for testing tools, performing network analysis, developing Python utilities, and exploring modern cybersecurity concepts in a controlled setting.
